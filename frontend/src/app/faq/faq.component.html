What is Shufflepik? Shufflepik is a website and a Discord bot.

Upload images or memes on the Shufflepik site and share a randomized image with the Shufflepik bot using the /shufflepik command on Discord.

How to upload images? Must have a Shufflepik account.

After creating a shufflepik account you follow 3 easy steps (in no particular order):

  1. If none of the server's you're a member of has the Shufflepik bot, invite Shufflepik Bot to one of your servers
  2. Connect Shufflepik to your Discord account
  3. Verify Email (sent to email used to create acct.)

After this follow the upload image prompt and you'll be able to upload images.

What do you mean "randomized" image? A random image

When you upload an image you will be prompted to upload to a corresponding "server pool". Any user from that server can upload images to the shared server pool on Shufflepik. The Shufflepik bot selects an image at random when a user uses the /shufflepik command. It's sort of a routlette of images.

Does a Discord server share a Shufflepik account? No

You will only be able to see images you upload on your Shufflepik account. You will only be able to see another user's images when they are displayed through /shufflepik command on Discord.

Compatible image extensions? WIP

Work in progress. We are only able to process .png, .jpg, .jpeg formats. We're working on gif compatibility, should be out soon.
